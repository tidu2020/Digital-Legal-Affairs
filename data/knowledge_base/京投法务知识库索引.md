# 国企数字法务系统 - 核心知识库索引

## 系统概述

国企数字法务（jingtou-legal）是国企法务合规咨询与审核支持的智能系统。

### 核心依据制度
- 《法律事务管理规定》
- 《诉讼仲裁案件管理办法》
- 《合同管理办法》
- 《合规管理规定》
- 《合规管理手册》

---

## "三个100%"要求

1. **重大经营决策法律审核率 100%**
2. **规章制度法律审核率 100%**
3. **合同法律审核率 100%**

---

## "四位一体"合规体系

法务、合规、内控、风控融合管理，支撑公司"十四五"战略落地和"三个转型"升级。

### 组织架构
- **党委**：统筹推进合规管理
- **董事会**：审议批准合规战略和基本制度
- **合规委员会**：统筹协调（与审计与风险管理委员会合署）
- **首席合规官（总法律顾问）**：全面负责，列席党委/董事会/总经理办公会
- **法律合规部**：合规管理综合部门，牵头"四位一体"融合管理
- **各部门/下属公司**：合规管理主责机构

## 关键时限要求

| 事项 | 时限 |
|------|------|
| 收到应诉文书 → 填报案件情况 | 3个工作日 |
| 下属企业拟提起诉讼 | 至少提前10个工作日 |
| 收到判决/裁定/调解书 | 3个工作日内报送 |
| 案件处理完结 → 归档 | 10日内 |
| 每季度结束后 → 更新案件信息 | 3个工作日 |
| 重大合同谈判 | 提前3个工作日通知法律合规部 |
| 合同履约评价 | 每年6月末和12月末各一次 |

## 重大案件标准

| 涉案金额 | 处理要求 |
|----------|----------|
| ≥1000万元 | 涉案责任部门主管领导 + 下属企业主要领导包案 |
| ≥5000万元 | 国企主要领导 + 下属企业主要领导包案 |

| 术语 | 含义 |

# 合规管理详细指南

## 一、"四位一体"合规融合体系

### 体系内涵
推进法务、合规、内控、风控"四位一体"融合建设，支撑公司"十四五"战略落地和"三个转型"升级，实现二级子企业合规管理体系建设全覆盖。

### 核心制度框架
**专项合规管理制度7项：**
1. 《合规管理规定》
2. 《法律事务管理规定》
3. 《内部控制及风险管理规定》
4. 《风险报告管理办法》
5. 《法治合规考核评价管理办法》
6. 《法律意见书管理办法》
7. 《法律合规审核实施细则》

**重点领域合规管理制度24项：**
《担保管理规定》《合同管理规定》《经营项目投资管理办法》《诉讼仲裁案件管理办法》《相关方安全管理办法》《数据资产管理办法》《个人信息处理全流程保护管理办法》等

### 机制保障
- 《总法律顾问工作规则》及合规联席会议管理机制
- 首席合规官（总法律顾问）明确职责
- 合规审核模块嵌入三大管理平台：项目评审、股权投资、合同审批
- 流程信息化管控

### 专项合规指引
- 《知识产权合规管理指引》
- 《数据安全合规管理指引》
- 《国际贸易合规指引》
- 《劳动用工专项指引》

## 二、合规管理组织体系及职责

| 主体 | 核心职责 |
|------|---------|
| 党委 | 统筹推进合规管理，研究重大事项，保障首席合规官履职，领导合规文化建设，监督董事会和高管履职，决定违规人员处理 |
| 纪委 | 维护合规制度权威，推动监察与合规衔接，监督合规落地，调查处理违规行为 |
| 董事会 | 审议批准合规战略、基本制度、年度报告，聘任首席合规官，决定合规机构设置，引领合规文化 |
| 合规委员会（与审计与风险管理委员会合署） | 统筹协调合规工作，审核合规规划/制度/年度报告，指导评价合规管理 |
| 经理层 | 建立健全合规组织架构，拟订合规制度，召集合规联席会议，培育合规文化，追究违规人员责任 |
| 工会 | 倡导合规文化，维护职工合规权益，审议通过相关合规事项，反映违规行为 |
| 首席合规官（总法律顾问担任） | 全面负责合规事务，列席党委/董事会/总经理办公会并发表意见，领导法律合规部，审批下属公司合规方案，考核下属首席合规官，发起合规调查，上报重大合规事项 |
| 法律合规部（合规管理综合部门） | 牵头"四位一体"融合管理，起草合规制度，开展风险识别评估、合规检查考核、合规信息化建设，负责法律审核与咨询，管理外部合规机构，受理违规举报，组织合规培训 |
| 其他部门/下属公司（合规管理主责机构） | 起草本领域合规专项制度，开展合规审查、风险排查，调查本领域违规事项，落实整改，向法律合规部报告风险 |
| 全体员工 | 合规履行岗位职责，接受合规培训，主动举报违规行为，寻求合规咨询 |

## 三、合规管理重点领域

### 重点领域清单
1. **公司治理** — "三重一大"决策、董事会治理、党委前置研究
2. **财务税收** — 预算管理、费用控制、税务合规
3. **劳动用工** — 劳动合同、薪酬福利、考勤管理、竞业禁止
4. **安全生产** — 安全生产责任制、事故报告和调查处理、消防安全
5. **知识产权** — 专利、商标、著作权、商业秘密保护
6. **数据安全** — 数据分级分类、个人信息保护、数据资产管理办法
7. **采购管理** — 7种采购方式适用规则、供应商管理、招投标合规
8. **投资管理** — 投资评审、境外投资、经营项目投资
9. **融资担保** — 担保审批权限、信息披露、融资风险管理
10. **国际合作** — 国际贸易合规、境外反垄断、出口管制

### 重点环节
- 制度制定环节
- 重大经营决策环节
- 重要合同审核环节
- 重大项目投资环节

### 重点人员
- 管理人员（尤其是"关键少数"）
- 重要风险岗位人员
- 海外人员

## 四、"三张清单"机制

### 1. 合规义务清单
明确"必须做什么、禁止做什么"的底线要求，覆盖：
- 国家法律法规义务
- 国资委监管要求
- 公司内部制度要求
- 行业规范要求

### 2. 流程控制清单
明确关键节点的审核标准和要求：
- 事项发起条件和前置审查
- 审批权限和流程节点
- 审核标准和风险阈值
- 异常情况处理机制

### 3. 岗位职责清单
明确各岗位的合规责任与失职后果：
- 岗位核心合规职责
- 合规红线和禁止行为
- 失职后果和追责标准
- 报告义务和时限要求

> 三类清单相互衔接、动态更新，要求员工"履职前看清单、决策时对照清单、完成后回顾清单"

## 五、合规管理内控流程

### 1. 制度制定流程
```
起草 → 主责机构初步审核 → 法律合规部法律+合规双审核 → 审批发布 → 动态更新
```

### 2. 重大经营决策流程
```
前期调研/尽职调查 → 主责机构初步审核 → 法律合规部审核
→ 重大事项首席合规官复核 → 合规联席会议/党委会/董事会/总经理办公会审议
→ 决策实施 → 风险跟踪
```

### 3. 合同管理流程
```
相对方遴选 → 起草 → 主责机构初步审核 → 法律合规部审核
→ 审批签署 → 履行 → 归档 → 纠纷处理
```

### 4. 风险报告流程
```
风险识别 → 主责机构上报 → 法律合规部审核
→ 分级处理（一般风险→合规联络员会议；重大风险→合规联席会议/决策会议）
→ 整改跟踪 → 年度报告
```

### 5. 合规检查流程
```
制定检查计划 → 开展检查（书面/现场）
→ 出具整改通知 → 主责机构提交整改方案
→ 整改实施 → 验收 → 结果纳入考核
```

## 六、员工核心合规义务

### 履职合规
- 诚信履职，不欺上瞒下、弄虚作假
- 审慎审查合同相对方资质
- 不签署仅依托合同管控风险或限制公司权益的合同

### 反腐败与反垄断
- 招投标领域禁止贿赂、超规格礼品与接待
- 遵守反垄断法规，触发审查时及时申报

### 财务合规
- 遵守财务制度，不伪造交易
- 不使用虚假票据侵占公司利益

### 知识产权保护
- 主动保护公司知识产权，及时申请注册、维护权利
- 侵权时及时维权

### 信息安全与保密
- 遵守信息披露制度，未经授权不得擅自对外披露
- 遵守保密规定，保守国家秘密、工作秘密和商业秘密

### 反洗钱
- 遵守反洗钱法规，充分了解客户和商业伙伴
- 避免与涉嫌洗钱的相对方交易

### 忠实义务
- 不得利用职务便利输送公司利益
- 不得投资竞争业务或为竞争者提供帮助

### 安全与环保
- 履行安全生产职责，不违章指挥、违规作业
- 遵守环保法规，做好环境影响评价和减排工作

> **公司对违规行为零容忍**，违规将按《干部员工惩处办法》追究责任。

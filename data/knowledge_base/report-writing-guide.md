# 国企法务合规公文写作指南

本文件为国企及下属企业法务合规类公文写作提供标准化模板和写作规范，覆盖7类核心公文。所有模板均蒸馏自国企实际公文，确保符合国企制度体系和行文规范。

---

## 一、年度法治工作总结报告

### 适用场景
国企各下属企业年度法治工作总结与下一年计划编制，按国企法律合规部通知要求报送。

### 标准结构

```
[公司全称]
[年度]年度法治工作总结

一、法治建设责任落实情况
  （一）顶层设计与战略引领
  （二）机构设置与资源保障
  （三）法治引领与文化建设
  （四）统筹体系运行与危机应对

二、法治工作组织体系建设情况
  （一）领导决策机构运行有效
  （二）法治工作机构设置健全
  （三）"三道防线"责任体系清晰
  （四）法治工作队伍稳步加强

三、法治建设工作开展情况
  （一）三个100%执行情况（重大决策/制度/合同审核率，附具体次数和数据）
  （二）案件管理情况（案件数、类型、阶段、影响评估、以案促改措施）
  （三）法治培训与宣传教育
  （四）制度建设与合同管理

四、合规、内控及风控工作开展情况
  （一）合规体系建设情况（试点进展、"三张清单"、专项指引）
  （二）合规体系运行情况（双流程审核、有效性自评）
  （三）内控管理工作（内控手册更新、自评价、缺陷整改）
  （四）风险管理工作（风险排查、重大风险识别、量化监测）

五、法治建设工作年度亮点
  （创新举措、体系突破、典型经验）

六、下一年度法治工作计划
  （聚焦方向、具体举措、目标指标）

[日期]
```

### 写作规范

| 规范项 | 要求 |
|--------|------|
| 总体风格 | 正式公文风格，严谨规范，无口语化表达 |
| 数据呈现 | "三个100%"必须附具体次数；案件管理附具体数据；内控缺陷附数量和类型 |
| 亮点写法 | 采用"一是……二是……三是……"分点列举，每点含创新做法+效果评估 |
| 计划写法 | 围绕"四位一体"框架展开，每项计划含目标+举措+预期成效 |
| 开篇范式 | 交代编制依据（国企通知要求），概述年度工作主线和总体成效 |
| 以案促改 | 案件描述后必须附管理改进措施，体现闭环管理 |
| 体系语言 | 使用"四位一体""三张清单""三道防线"等国企体系术语 |

### 关键数据必填项
- 重大决策参与率/意见率（次数+百分比）
- 制度法律审核率（签批份数+百分比）
- 合同/协议法律审核率（审核份数+出具法律意见书份数+百分比）
- 案件数量及类型（仲裁/诉讼、原告/被告、涉案金额、当前阶段）
- 法治培训次数、参与人次、形式
- 合规体系成果（几清单、几指引、几制度）
- 内控评价缺陷数量（设计缺陷N项+执行缺陷N项）
- 风险库风险事项总数、重大风险项数

## 二、法律合规意见

### 适用场景
国企法律合规部针对重大经营决策、投资提案、项目方案调整等事项出具法律合规审核意见。

### 标准模板

```
国企法律合规部

国企法律部法学[年份]XX号

关于[事项名称]的法律合规意见

根据《[法律1]》《[法律2]》《[规范性文件]》等法律法规及规范性文件，
结合经办部门提供的相关资料出具日之前发生的有关事实和公司管理制度，
现就[事项名称]（[简称]）出具如下法律合规意见：

一、[主体资格符合性 / 事项基本情况]
  （说明事项背景、主体适格性、交易结构等）

二、[事项涉及的内外部审批程序]
  （根据公司制度明确需履行的审批层级：
   总经理办公会→党委会→董事会等）
  引用具体制度：《党委会议事规则》《贯彻落实"三重一大"决策制度的实施办法》
  《董事会工作规则》等

三、提请注意事项
  （风险提示、程序要求、后续需关注事项）

综上所述，法律合规部认为，[结论性意见，格式为：
"上述请示事项不存在违反法律法规、相关国资监管政策的情形，
可以依照相关制度要求依法合规开展/履行/推进实施。"或
"该提案符合相关法律法规及公司制度，可依法合规推进。"]

（以下无正文，为《关于[事项名称]的法律合规意见》的签字盖章页。）

总法律顾问：[签名]

（联系人：[姓名]；联系电话：[电话]）

抄送：[部门1]，[部门2]，存档。

国企法律合规部
[日期]印发
```

### 写作规范

| 规范项 | 要求 |
|--------|------|
| 文号格式 | 国企法律部法学[年份]XX号 |
| 法律依据 | 开篇列明所有适用法律法规及规范性文件，按效力层级排列 |
| 结论用语 | 必须明确肯定或否定，禁用模糊表述；肯定用语："可依法合规推进/履行/开展" |
| 风险提示 | "提请注意事项"部分不回避风险，但用语客观中立 |
| 制度引用 | 审批程序必须引用具体制度名称和条款 |
| 签章页 | "以下无正文"后附签字盖章页格式，总法律顾问签名 |

## 三、合规管理工作季报

### 适用场景
国企本部各部门及下属企业按季度填报合规管理工作情况，通过OA《国企-合规管理工作季报流程》提交。

### 填报结构与内容

```
一、合规管理
  1. 建设情况：本单位"四位一体"合规管理创新举措、重大进展等
  2. 行业要闻：外部单位合规经验、处罚等
  3. 新法速递：各单位结合各自业务查找新政策法律法规，研判对公司业务影响

二、风险收集
  填写：风险名称、风险分类、风险划分、发生时间、风险描述、
  产生原因、公司自身缺陷问题查找/管理启示、应对举措

三、国企年度重大风险防控落实情况
  根据年度《风险管理工作报告》中确定的重大风险，
  选取风险名称和应对举措，填写本季度化解进展

四、其他
  1. 季度法律纠纷风险排查情况（附件）
  2. 京外企业经营投资风险防控专项治理风险指标监测情况（附件，涉及京外业务单位）
  3. 内控专项整改落实情况（附件，仅填未完成或近期完成事项）
```

### 填报规范

| 规范项 | 要求 |
|--------|------|
| 填报路径 | OA平台→流程→新建流程→《国企-合规管理工作季报流程》 |
| 零报告 | 除"年度重大风险防控落实情况"外，无报告事项选"零报告"或填"无" |
| 不重复填报 | 涉及《风险报告管理办法》重大风险的走《国企-企业重大风险情况报告》流程；涉及诉讼案件的在法务诉讼仲裁系统填报 |
| 审批流程 | 本部：部门风管人员→部门负责人→法律合规部人员→法律合规部负责人<br>下属公司：风管人员→风管部门负责人→法律合规部人员→法律合规部负责人 |
| 风险事项 | 个别风险事项层报公司领导：总法律顾问→主管领导→总经理→董事长 |
| 同步报送 | 法律纠纷风险排查与合规季报同步报送 |

## 四、风险管理工作年度报告

### 适用场景
国企向董事会提交的年度风险管理工作报告，总结全年风险管理工作并规划下年度工作。

### 标准结构

```
一、风险管理整体情况
  （一）总体风险战略与偏好
  （二）风险管理治理机制运行情况

二、统筹推进"四位一体"综合风险防控体系建设情况
  （一）法律审核关口前移，注重风险过程管理
  （二）推动风险内控协同，从严开展专项治理
  （三）案件管理减存控增，以案促鉴效果显现
  （四）铺开合规体系建设，强化集团管控质效

三、上一年度重大风险化解情况
  （一）多措并举化解重大风险，抓实抓牢提升防范能力
  （二）系统协力处置重大案件，以案促管提高管理质效
  （三）及时上报重大风险事件，分类施策加速跟踪处置
  （四）促进重点领域合规建设，主动前瞻专项风险防范

四、全面风险管理评估工作情况
  （一）风险信息搜集辨识信息化、动态化
  （二）风险清单和分析框架优化、细化
  （三）风险评估工作专业化、准确化
  （四）风险评估结果综合化、导向化

五、本年度重大风险解决方案
  （每项重大风险按"风险现状及趋势变化"+"风险降低与化解措施"展开，
   措施明确主责单位）

六、下一年度风险管理工作计划
```

### 写作规范

| 规范项 | 要求 |
|--------|------|
| 整体风格 | 正式公文，提交董事会审议，属机密级文件 |
| 逻辑框架 | "总-分-总"：先总述背景和成效，再分模块详述，最后规划 |
| 数据呈现 | 案件数据（存量/新发/结案/涉案金额/同比变化）、风险评估问卷发放与回收数据、加权评估结果 |
| 对比分析 | 年度对比突出变化，如"新发案件数量同比增长X%，涉案金额同比下降Y%" |
| 风险解决方案 | 每项重大风险列"现状+措施+主责单位"，措施逐条编号 |
| 权责明确 | 所有举措标注主责单位，落实到具体部门/下属企业 |
| 客观中立 | 既肯定成效，不回避问题（如"持续亏损""事故仍时有发生"） |

### 关键数据必填项
- 存量案件数、新发案件数、结案数、结案率、胜诉率
- 涉案金额及同比变化
- 执行收回金额、避免/挽回损失金额
- 风险评估问卷发放/回收数、覆盖范围
- 重大风险数量及与上年度对比
- 法律意见书出具份数、OA审核合同份数、制度审核份数
- 合规试点单位覆盖情况

## 五、内控体系工作报告

### 适用场景
国企向董事会提交的年度内控体系工作报告，由审计与风险管理委员会审议后报董事会批准。

### 标准结构

```
一、汇报背景说明
  （编制依据、报告审批情况、提请对象）

二、[年度]内控体系建设情况
  （一）内控工作领导体制建设情况
    1. 加强组织领导，落实治理层主体责任
    2. 抓好"关键少数"，强化"第一责任人"职责
    3. 集团统筹协调指导，推动内控体系全覆盖
  （二）内控工作组织架构及履职情况
  （三）内控体系建设与执行情况
    1. 常态化开展更新，认真梳理制度流程
    2. 对标本部制度，子企业优化自身制度
    3. 健全境外内控制度，严格执行境外制度
    4. 推动新业务单元制度建设
    5. 强化日常监督及专项治理
    6. 聚焦重要业务领域，加强关键环节管控
  （四）信息化管控情况
  （五）其他方面情况

三、内控体系监督评价工作情况
  （评价机制、自评价情况、监督评价发现的问题、结果运用）

四、内控缺陷整改落实情况

五、内控工作取得的成效和亮点

六、内控工作下一步工作安排
  （一）加强内控组织领导，健全体制机制
  （二）持续完善内控制度体系，确保有章可循
  （三）提升重点领域内控运行，有效防控风险
  （四）加强信息化建设，提高内控信息化管控水平
  （五）抓实内控监督评价，以评促建、举一反三
  （六）开展内控培训宣贯，提升风险防控意识

七、附件
  附件1：内部控制缺陷认定汇总表
  附件2：内控突出问题情况表
  附件3：内控缺陷情况表
```

### 写作规范

| 规范项 | 要求 |
|--------|------|
| 开篇范式 | 明确编制依据（市国资委文件号）、审批流程（总经理办公会审议通过）、提请对象 |
| 问题导向 | 不回避短板，单设"监督评价发现的问题""缺陷整改"章节 |
| 体系化表述 | 围绕"四位一体"融合思路，遵循"体系建设-监督评价-缺陷整改-成效总结-未来规划"闭环 |
| 分层分类 | 子企业、不同业务领域按层级和类型拆分说明 |
| 数据呈现 | 制度新增/修订/废止数量、内控流程修订数、缺陷总数及分类、子企业覆盖率 |

## 六、案件风险防范与化解专项合规报告

### 适用场景
事业部或下属单位就案件风险防控体系建设的专项合规报告，分析案件管理框架、应对流程和整改措施。

### 标准结构

```
一、案件专项合规的背景
  1.1 国有企业作为市场主体难以避免案件发生
  1.2 加强案件管理是法治国企建设的基本要求
  1.3 案件专项合规是提高企业案件管理水平的有力手段
  1.4 历史案件情况提示亟需开展案件专项合规

二、案件专项合规的目标
  2.1 总目标
  2.2 事业部层面目标（整合现有工具/规范处理流程/以案促改）
  2.3 更高层次目标（落实国资委要求/推动法治国企建设）

三、案件风险防范与化解的基本框架
  日常风险监控 → 潜在风险化解 → 涉诉纠纷解决
  （基础环节）   （前置缓冲）     （最终应对）

四、日常风险监控（第一道防线）
  4.1 业务部门实时监控
  4.2 合同部门履约评价
  4.3 法务部门跟进关注

五、潜在风险化解（第二道防线）
  5.1 内部化解路径
  5.2 外部合力化解
  5.3 法律纠纷风险排查与上报

六、涉诉纠纷解决（第三道防线）
  6.1 案件应对策略选择
  6.2 外聘律师管理

七、案件的法学分析方法及基本应对流程
  7.1 法律关系分析法
  7.2 请求权基础分析法
  7.3 案件基本应对流程（五步法）

八、针对案件的整改（事后闭环）
  8.1 提出改进措施
  8.2 增补合规指引
  8.3 加强合规培训

九、附件
  附件一：既有案件情况汇总（按案件类型分表格）
```

### 写作规范

| 规范项 | 要求 |
|--------|------|
| 逻辑主线 | "背景-目标-框架-方法-整改"完整闭环 |
| 三层防控 | 前端防控（日常监控）→中端处置（潜在风险化解）→后端应对（案件处理+整改） |
| 政策引用 | 明确标注文件名称和文号，如国资委〔2021〕80号、京国资发〔2023〕26号 |
| 法学方法 | 案件分析采用"法律关系分析法"+"请求权基础分析法"双法互补 |
| 案件数据 | 附件表格按案件类型统计总数、仲裁/一审/二审/调解/再审分项数 |
| 整改闭环 | 案件后必须"提出改进措施→增补合规指引→加强合规培训"三步闭环 |

## 七、法律纠纷风险排查通知

### 适用场景
国企法律合规部发布法律纠纷风险排查工作通知，指导本部及下属企业开展年度风险排查。

### 标准模板

```
国企法律合规部

关于开展[年度]年度公司法律纠纷风险排查工作的通知

公司各部、下属公司：

为贯彻落实市国资委案件管理政策精神和法治考核的具体要求，
推动公司法律纠纷风险排查机制有效落地，根据公司《诉讼仲裁案件管理办法》，
法律合规部现组织开展[年度]年度法律纠纷风险排查工作，
具体工作安排如下：

一、排查范围
  公司本部、事业部及公司各级全资和控股企业及
  前述企业出资设立的拥有实际控制权的有限合伙企业。

二、排查事项
  排查范围内各部门及下属公司[年度]年度可能发生法律纠纷风险的事项
  （不含已形成诉讼或仲裁的案件）。

三、工作原则及要求
  1. 及时预警。按照"预防为主、调处结合"的原则……
  2. 完善机制。建立法律纠纷风险管理台账，动态更新，通过"合规季报"报送……
```

## 通用写作原则

### 行文风格
1. **正式公文体**：严谨规范，杜绝口语化、网络用语
2. **数据说话**：每个结论有数据或实例支撑，避免空泛表述
3. **逻辑闭环**：从问题识别到措施落地形成完整管理闭环
4. **权责清晰**：举措必须标注主责单位/部门，可考核、可追溯
5. **体系联动**：体现"四位一体"（法务+合规+内控+风控）融合思路

### 常用体系术语
| 术语 | 含义 |
|------|------|
| 四位一体 | 法务、合规、内控、风控融合管理 |
| 三个100% | 重大决策法律审核率100%、规章制度法律审核率100%、合同法律审核率100% |
| 三张清单 | 合规义务清单、流程控制清单、岗位职责清单 |
| 三道防线 | 第一道（业务部门）→第二道（法律合规部门）→第三道（监督/审计部门） |
| 减存控增 | 减少存量案件、控制新增案件 |
| 以案促管/以案促改 | 通过案件分析发现管理漏洞、推动管理改进 |
| 双重预防机制 | 安全风险分级管控+隐患排查治理 |
| 压存控增、提质创效 | 国资委案件管理专项行动口号 |
| 1+N | 1个基本制度+N个专项指引/实施细则 |

### 数据呈现规范
- 绝对值+同比/环比变化：如"新发案件30件，同比增长3.45%，涉案金额4.15亿元，同比下降28.32%"
- 百分比+具体次数：如"制度法律审核率达到100%，签批审批单14份"
- 分类统计：按类型/层级/领域拆分，配合表格展示
- 趋势判断：先列数据，再说明趋势和原因

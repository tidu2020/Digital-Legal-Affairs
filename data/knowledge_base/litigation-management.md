# 诉讼仲裁案件管理详细指南

## 一、案件分类与管辖

### 案件范围
- 国企本部及各下属企业在经营管理过程中发生的，以本公司或各下属企业作为一方当事人的各类诉讼及仲裁案件

### 案件分类
| 类型 | 处理主体 | 管理方式 |
|------|---------|---------|
| 国企为涉案主体 | 法律合规部牵头 + 涉案责任部门配合 | 本部案件处理方式 |
| 线路公司为涉案主体 | 原则参照本部案件处理 | 由下属公司或委托处理 |
| 下属企业为涉案主体 | 下属企业自行处理 | 法律合规部指导监督 |

### 涉案责任部门确定
- 公司本部作为当事人 → 与纠纷事项直接相关的部门
- 下属企业作为当事人 → 该下属企业的本部业务主管部门
- 特殊情况 → 经法律合规部与相关部门研究，报总法律顾问、公司领导批准后指定

## 二、国企/线路公司案件处理流程

### 被诉案件（作为被告/被申请人/第三人）

```
收到应诉文书
    ↓ 3个工作日内填报案件情况
法律合规部确定涉案责任部门
    ↓ 转交法律文书副本
涉案责任部门确认案件情况
    ↓
法律合规部研提初步应对建议
    ↓ 报请
涉案责任部门分管领导 → 总法律顾问 → 法律合规部分管领导 → 总经理审核 → 董事长批准
    ↓
委托关联单位代理的，书面发出委托函
（含代理范围、处理基本原则、诉讼期间联系人机制）
```

### 主诉案件（作为原告/申请人）

```
拟提起诉讼/仲裁
    ↓ 至少提前10个工作日填报起诉申请
    ↓ 注意：最迟应在诉讼时效到期前2个月
法律合规部进行形式审查
    ↓ 研提意见建议
报涉案责任部门分管领导 → 总法律顾问 → 法律合规部分管领导审批
    ↓
组织起诉
```

## 三、下属企业案件处理流程

### 被诉案件
1. 第一时间在诉讼仲裁案件管理系统填报案件情况
2. 确定案件承办人员
3. 收到法律文件后3个工作日内填报案件相关情况
4. 法律合规部研提意见建议，报涉案责任部门分管领导、总法律顾问、法律合规部分管领导审批

### 主诉案件
1. 至少提前10个工作日填报起诉申请（诉讼时效到期前2个月）
2. 属重大案件的，应将审批时间一并预留
3. 法律合规部形式审查 → 研提意见 → 审批

### 判决/裁定后
- 收到判决书、裁定书、调解书、仲裁裁决书 → 3个工作日内报送扫描件并填报下一步程序
- 下属企业对诉讼时效负责

## 四、重大案件管理

### 重大案件标准
- 涉案金额1000万元以上的案件

### 重大案件处理要求
1. 按规定时限填报，附案件解决时间表及任务分解表
2. 案发之日起5个工作日内形成案件详细情况报告提交法律合规部
3. 法律合规部报告案件情况及意见建议
4. 按制度要求报公司党委会前置研究 → 总经理办公会审批
5. 定期向董事会报告
6. 按市国资委要求备案
7. 法律合规部全程跟进（专题会、现场指导、定期听取汇报、指导律师管理使用）

### 领导包案制
| 涉案金额 | 包案领导 |
|---------|---------|
| 1000万元以上 | 涉案责任部门主管领导（国企领导班子成员）+ 涉案下属企业主要领导 |
| 5000万元以上 | 国企主要领导 + 涉案下属企业主要领导 |
| 国企本部重大案件 | 国企领导 + 涉案责任部门领导 |

包案领导定期听取案件进展，督促、指导案件处理工作。

### 历史遗留案件
- 特别是主责部门或下属企业领导变更的，应给予重视
- 定期向法律合规部报送案件处理进展
- 法律合规部加强指导

## 五、外聘律师管理

### 选聘规则
- 处理诉讼仲裁案件一般应聘请律师代理
- 公司本部案件：法律合规部归口管理选聘
- 下属企业案件：下属企业自行负责
- 重大法律纠纷案件：代理律师原则上从本部合格供应商名录选聘
- 选聘按《外聘法律顾问管理办法》执行

### 律师监督
- 法律合规管理部门及时与律师沟通案件情况
- 研究诉讼对策，掌握案件进展
- 对律师工作进行全程监督
- 未尽职律师：及时解除委托代理协议 → 取消律所合格供应商资格 → 依法追究法律责任

## 六、案件统计与档案管理

### 统计分析
- 法律合规部及下属企业建立诉讼案件管理台账
- 定期统计分析：案件特点、原因、暴露的业务风险点和管理漏洞
- 有针对性地提出预防和改进建议

### 季度更新要求
- 每季度结束后3个工作日内集中核查案件信息更新
- 确认案件信息填报至最新状态
- 完成季度案件信息更新确认
- 未结重大案件单独填写最新进展

### 档案管理
- 案件处理完结后10日内线上线下同步整理归档
- 国企/线路公司案件 → 法律合规部归档 → 提交公司办公室等档案管理部门
- 下属企业案件 → 该下属企业归档

**案件档案包括但不限于：**
- 案件受理通知书、应诉通知书
- 起诉状、仲裁申请书
- 相关证据
- 答辩状、代理词
- 仲裁决定书、裁定书、判决书
- 其他过程性材料

## 七、罚则

- 违反本办法规定的期限、流程 → 法律合规部通过考核、通报、公示等方式批评
- 造成较大经济/名誉损失 → 按公司制度追究经办人员和相关领导责任
- 构成犯罪 → 移交司法机关处理

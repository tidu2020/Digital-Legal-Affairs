# 制度法律审核详细指南

## 一、审核范围

京投公司管理制度法律审核覆盖公司本部、所属公司、直管公司、控股公司、参股公司外派人员相关管理，以及有实际控制力的下属企业的各类制度。涵盖新增制度和修订制度两类。

### 制度分类与典型示例

| 类别 | 典型制度 |
|------|---------|
| 公司治理类 | 《公司章程》《总经理工作规则》《董事会授权管理暂行办法》、"三重一大"决策制度实施办法 |
| 业务管理类 | 《经营类投资项目评审管理办法》《境外投资管理办法》《融资管理规定》《轨道交通资产管理办法》《经营性房产租赁经营管理规定》 |
| 人力资源类 | 《考勤管理办法》《人才周转房管理办法》《劳动合同管理细则》《员工职级晋升管理办法》《中层领导人员管理办法》 |
| 财务管理类 | 《财务预算管理规定》《费用报销审批管理办法》《国有资产评估管理规定》《政府专项债券管理办法》 |
| 安全与保密类 | 《安全生产管理规定》及配套安全管理制度、《保密管理规定》及保密管理办法、《数据资产与数据安全管理办法》 |
| 党建纪检类 | 《党委会议事规则》《巡察工作办法》《纪检监察机构监督执纪执法工作办法》 |
| 信息化与采购类 | 《采购管理规定》《合同管理办法》《招标实施细则》《电商直采实施细则》 |
| 科研与创新类 | 《科研管理规定》《科研项目管理办法》《科技成果转化管理办法》《科技创新奖励办法》 |

## 二、审核全流程

```
制度起草 → 征求意见 → 法律审核申请 → 法律合规部审核 → 决策审批 → 发布备案
```

### 1. 制度起草与发起
- 主责部门根据外部监管要求、公司管理需要、组织架构调整等情况完成初稿
- 明确制度定位（新增/修订）、适用范围、职责分工、核心管理流程
- 非原则性修订（如部门名称调整、试行转正式）可简化流程

### 2. 征求意见
**征求意见范围：**
- 公司本部相关部门：法律合规部、公司办公室、党委组织部、纪监部门、审计事务部、财务管理部、人力资源部、合同管理部、信息数据管理部等
- 下属单位：轨道公司、轨道运营公司、装备集团、路网公司、土地开发事业部、资产经营开发事业部、京投科技、管廊公司等
- 特殊相关方：工会系统（涉及职工权益类制度）、职工代表团、基层工会

**征求意见方式：** 发函征求意见、召开会议研讨、线下沟通对接

**意见处理规则：**
1. 对合理建议直接采纳，同步修改对应条款，标注修订内容
2. 对不符合公司实际或与现有制度冲突的建议，与提出方沟通解释，达成一致后说明未采纳原因
3. 涉及多部门职责的争议事项，组织专题会议研讨，或提交分管领导协调确定

### 3. 法律审核申请
提交材料包括：
- 制度最终稿
- 征求意见汇总及采纳情况
- 制度修订说明（修订背景、核心修订内容）
- 主责部门负责人、经办人、联系方式
- 制度属性标注（新增/修订、制度层级）

### 4. 法律合规部审核
**审核维度：**
- 合法性：与现行法律法规、国资委规定、公司章程的一致性
- 合规性：符合合规管理体系要求，嵌入合规控制点
- 风险可控性：审批权限明确、流程闭环、责任可追溯
- 体系兼容性：与现有制度体系不冲突，职责边界清晰，名词定义准确
- 制度法律+内控双审核机制

### 5. 决策审批
- **一级制度、涉及重大经营管理事项**：党委会、董事会、总经理办公会审议
- **二级、三级制度**：总经理办公会或相应主管领导审批
- **非原则性修订**：主责部门履行签报审批程序后纳入制度体系

### 6. 发布与备案
制度通过审批后正式发布，纳入公司制度体系，同步向相关部门、单位备案，后续按规定开展宣贯、执行监督、定期评估修订。

## 三、各部门核心审核要点

| 部门 | 核心关注方向 |
|------|------------|
| 法律合规部 | 合法性合规性、风险管控、审批流程权限、名词定义准确性、与现有法律合规制度衔接、境外项目合规审核要求 |
| 公司办公室 | 文字表述规范性、行文逻辑、与公司治理类制度衔接、部门名称一致性、公文格式合规性 |
| 审计事务部 | 与内控/风险管理体系衔接、审计监督权限、缺陷认定标准、评价结果应用 |
| 人力资源部 | 职工权益合规性、干部管理、薪酬福利、绩效考核、人员调配 |
| 财务管理部 | 财务审批流程、预算衔接、经费标准、财务风险控制 |
| 下属单位 | 与业务实际适配性、职责分工合理性、流程可操作性 |

## 四、典型制度审核案例

### 案例1：《采购管理规定》及6项采购实施细则
- **类型：** 新增+修订
- **征求意见：** 法律合规部、合同管理部、公司办公室、10余家下属单位，征集100余条建议，采纳约60条
- **核心修订：** 优化采购管理模式、明确7种采购方式适用规则（招标、比选、竞争性谈判、单一来源、询价、合格名录直采、电商直采）、细化审批权限、建立供应商管理体系

### 案例2：《合规管理规定》《法律合规审核实施细则》
- **类型：** 新增+修订
- **征求意见：** 各部门及单位书面意见24份，有效意见110余条，采纳大部分
- **核心修订：** 搭建合规管理体系、明确法律合规审核100%覆盖要求、完善审核流程

### 案例3：121项非原则性调整制度（2021年4月）
- **背景：** 组织架构调整后的部门名称、职务名称更新，试行制度转正式版本
- **处理方式：** 集中审核，已全部征求各部及所属公司意见并沟通一致

## 五、制度分级说明

| 层级 | 审批权限 | 典型制度 |
|------|---------|---------|
| 一级制度 | 党委会/董事会/总经理办公会 | 《公司章程》《合规管理规定》《内部控制及风险管理规定》 |
| 二级制度 | 总经理办公会 | 各业务领域管理办法 |
| 三级制度 | 主管领导审批 | 具体实施细则、操作规程 |

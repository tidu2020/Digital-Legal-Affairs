## 二、合同全流程详解

### 1. 合同需求提出
- 需求部门提出合同需求，经公司批准
- 如需通过采购产生，按采购流程执行

### 2. 确定合同相对方
- 承办部门负责确定合同相对方
- 相对方资质审查：营业执照、法定代表人身份证明
- 如为分公司签约，需出具法定代表人授权签订合同的证明原件或其他有效证明

### 3. 合同谈判
- 需法律合规部、财务管理部或其他部门参加的，一般提前3个工作日书面通知
- 提供背景材料和谈判内容大纲

### 4. 合同文本起草
- 承办部门负责起草，法律合规部和合同管理部参与
- 签订合同应以公司名义采用书面形式

### 5. 合同审核（会签流程）
```
承办部门发起 → 部门负责人签署 → 相关部门会签 → 财务管理部审核 → 法律合规部审核 → 合同管理部审核 → 领导审批
```

**提交审核的材料：**
- 批准合同事项的签报、会议纪要等合同订立依据
- 对方当事人相关证明材料或其他相关材料

### 6. 签署用印
- 一般加盖合同专用章，由承办人办理施印手续
- 必须使用公章或法人手签章时，按印信管理办法执行

### 7. 合同签署后归档
- 公司至少留存两份原件
- 一份交合同管理部收存（附《合同审批表》《授权委托书》）
- 另一份由履约部门管理
- 承办部门在合同管理系统上传合同原件扫描件

### 8. 合同履行
- 履约部门负责履行监控和支付管理
- 履约评价：每年度6月末和12月末最后5个工作日内各一次
- 合同支付严格按约定执行，严禁超合同金额支付
- 合同管理部参加其组织采购的合同项目支付审核

### 9. 合同变更与纠纷
- 合同变更须采用书面形式
- 需签订补充协议的，按同等审核流程办理
- 合同纠纷由法律合规部牵头处理

## 三、合同审核清单

### 法律合规部审核要点
- [ ] 合同主体资格和资质审查
- [ ] 合同条款完整性（标的、数量、质量、价款、履行期限和方式、违约责任、争议解决）
- [ ] 权利义务对等性
- [ ] 违约责任条款的合理性和可执行性
- [ ] 争议解决方式（仲裁/诉讼，管辖约定）
- [ ] 知识产权相关条款
- [ ] 保密条款
- [ ] 不可抗力条款
- [ ] 合同生效条件和签署要求
- [ ] 与法律法规、公司制度的合规性
- [ ] 保证金/担保条款合规性
- [ ] 合同解除和终止条款

### 财务管理部审核要点
- [ ] 付款条件和付款方式
- [ ] 发票开具要求
- [ ] 税费承担约定
- [ ] 预算匹配情况
- [ ] 结算方式

### 合同管理部审核要点
- [ ] 合同编号规范性
- [ ] 合同文本格式规范性
- [ ] 合同分类和归档要求
- [ ] 合同履行期限和节点设置
- [ ] 合同范本使用情况

## 五、合同范本管理

- 合同管理部根据业务特点组织制定合同范本
- 合同范本原则上由公司总经理办公会通过后实行
- 合同管理部根据业务部门需求不断充实完善标准文本库
- 各部门总结合同条款执行中的问题，提出改进建议
- 合同范本原则性调整 → 公司总经理办公会审批
- 合同范本非原则性调整 → 公司总经理审批

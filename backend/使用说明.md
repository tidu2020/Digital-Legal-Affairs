## 🚀 启动服务的方法

### 方法 1: 使用批处理文件（推荐）
双击运行 `启动服务.bat`

### 方法 2: 使用 Python 脚本
```bash
python direct_start.py
```

### 方法 3: 使用 uvicorn 命令
```bash
python -m uvicorn app:app --host 0.0.0.0 --port 1824
```

### 方法 4: 直接运行 app.py
```bash
python app.py
```
